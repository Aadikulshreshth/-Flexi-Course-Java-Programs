# Java Programming Lab – OOP Practice

This repository contains Java console programs developed as part of academic lab practice.
The programs focus on strengthening core programming logic and understanding of Object-Oriented Programming (OOP) concepts.

## Topics Covered

* Classes and Objects
* Constructors
* Encapsulation
* Methods and Modular Programming
* Arrays and ArrayList
* Exception Handling (including custom exceptions)
* Menu-driven console programs
* User input using Scanner
* Control flow (loops, switch-case, conditionals)

## Repository Structure

Each Java file or folder represents an individual program based on a specific problem statement or concept.

New programs may be added over time as part of continued lab work and practice.

## How to Run a Program

Compile:
javac FileName.java

Run:
java FileName

## Purpose

The aim of this repository is to build a strong foundation in Java programming by implementing small console-based applications and practicing object-oriented design principles.
